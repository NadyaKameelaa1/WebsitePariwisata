# Selamat datang di projek Website Pariwisata saya!
Repositori ini dibuat untuk keperluan projek Website Pariwisata yang saya beri nama "Hello Indonesia". Proyek ini dikembangkan sebagai platform informasi destinasi wisata di daerah Bali dengan lima kategori utama yaitu Gunung, Pantai, Kuliner, Sejarah, dan Budaya. Melalui website ini, pengguna dapat menjelajahi berbagai destinasi wisata, melihat detail informasi dan gambar, serta menambahkan tempat favorit setelah melakukan login. Sistem ini juga menyediakan role Admin, yang memiliki akses ke Dashboard untuk mengelola data wisata melalui fitur CRUD, mengatur gambar destinasi, menampilkan statistik favorit bulanan dalam bentuk grafik, serta memantau riwayat aktivitas pengguna secara real-time. Dengan desain antarmuka yang modern dan integrasi backend yang terstruktur, Hello Indonesia diharapkan dapat menjadi contoh penerapan teknologi web yang informatif, responsif, dan bermanfaat bagi wisatawan maupun pengelola data pariwisata.
